# Nordik Pro - Sitio Web Oficial

Sitio web moderno y épico para Nordik, construido con **React** + **Vite** + **Framer Motion** + **CSS3**.

## 🚀 Características

- ✨ Animaciones épicas con Framer Motion
- 🎨 Diseño minimalista futurista con SVGs lineales
- 💻 Responsive design (Mobile, Tablet, Desktop)
- 🚄 Rendimiento optimizado con Vite
- 🎯 Colores Nordik (Azul #009DFF, Fucsia #E71CEB)
- 📱 Componentes reutilizables
- ✦ Efectos de partículas animadas
- 🎭 Transiciones suaves en todas partes

## 📋 Requisitos

- Node.js 14+ 
- npm o yarn

## 🛠️ Instalación

1. Navega a la carpeta del proyecto:
```bash
cd nordik-support
```

2. Instala las dependencias:
```bash
npm install
```

## 🚀 Desarrollo

Para ejecutar el servidor de desarrollo:

```bash
npm run dev
```

El sitio estará disponible en `http://localhost:5173`

## 🏗️ Build

Para construir para producción:

```bash
npm run build
```

Para previsualizar la build:

```bash
npm run preview
```

```
│   │   ├── Footer.jsx          # Pie de página

│   └── main.jsx                # Punto de entrada

└── package.json               # Dependencias
```

## 💡 Tips de Desarrollo

- Las animaciones usan Framer Motion para mejor rendimiento
- Los SVGs lineales son totalmente personalizables
- Los colores están en variables CSS `:root`
- Las animaciones pueden ajustarse en los componentes individuales
- El diseño es 100% responsive

## 🚀 Deploy

Para deployar a Vercel, Netlify o similar:

1. Push a tu repositorio Git
2. Conecta con tu platform de hosting
3. Selecciona `npm run build` como comando de build
4. Deploy!
