🚀 CÓMO USAR

1. INSTALAR
   cd nordik-support
   npm install

2. DESARROLLO
   npm run dev
   → Se abre en http://localhost:5173

3. COMPILAR
   npm run build
   → Crea carpeta 'dist' lista para deploy

4. DEPLOY
   Vercel:   Conectar GitHub + Deploy automático
   Netlify:  Mismo proceso
   Servidor: Copiar 'dist' a /var/www/
