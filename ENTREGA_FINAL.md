## 🚀 Cómo Usar

### 1. Instalación (2 minutos)
```bash
cd nordik-support
npm install
```

### 2. Desarrollo (instantáneo)
```bash
npm run dev
# Se abre en http://localhost:5173
```

### 3. Build para Producción (1 minuto)
```bash
npm run build
# Carpeta 'dist' lista para deploy
```

### 4. Deploy (5 minutos)
- Vercel: Conectar GitHub + Deploy automático
- Netlify: Mismo proceso
- Servidor propio: Copiar 'dist' a /var/www/

## 🆘 Soporte Rápido

### Problema: No se ve el logo
**Solución:** El logo está en `/public/logo.png`, asegúrate que exista

### Problema: Sitio muy lento
**Solución:** Ejecuta `npm run build` para una versión optimizada

### Problema: Formulario no funciona
**Solución:** Necesitas conectar un backend (ver BACKEND_INTEGRATION.md)

### Problema: Colores no son los correctos
**Solución:** Revisa `/src/index.css` en la sección `:root`
